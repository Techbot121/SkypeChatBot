# Skype Chatbot #

### How do set it up? ###

* Get [Visual studio](visualstudio.com) or similar.
* Build the Solution
* Run it

####Note:
- If you have Skype not running, the bot will launch it for you.
- Skype *WILL* ask if you want to run it first. It looks like this
